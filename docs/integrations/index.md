# Integrations

Kreuzberg provides official integrations for popular tools and databases, allowing you to easily plug our powerful document extraction pipeline into your existing workflows.

Available integrations:

- [Langchain](langchain.md): Seamlessly extract text from 75+ document formats into LangChain `Document` objects for RAG and AI pipelines.
- [SurrealDB](surrealdb.md): Directly ingest documents into SurrealDB with automated schema management, chunking, embeddings, and hybrid search.

More integrations are coming soon!
